# Backlog MVP

- Como usuario, quiero subir varias imágenes en un lote para optimizarlas rápidamente.
- Como usuario, quiero elegir un preset para ajustar calidad y ancho máximo.
- Como usuario, quiero ver el estado y progreso de cada job.
- Como usuario, quiero descargar un ZIP con los archivos optimizados.
- Como usuario, quiero ver un reporte antes/después por archivo.
